# ☕ Coffee Card UI

Uma interface moderna e minimalista de cartão de produto desenvolvida com Flutter, focada em design limpo, organização de layout e experiência do usuário.

---

## 🚀 Sobre o projeto

O Coffee Card UI é um projeto simples, porém estratégico, criado para praticar construção de interfaces no Flutter utilizando componentes do Material Design.

A aplicação apresenta um card de produto (Cappuccino), contendo imagem, descrição, preço e um botão de ação, simulando um cenário real de aplicativo de cafeteria ou e-commerce.

> Um projeto pequeno no tamanho, mas grande na intenção: clareza visual, código limpo e base para expansão.

---

## 🎯 Objetivo

- Praticar desenvolvimento de UI no Flutter
- Trabalhar com layout usando Column, Container e Padding
- Aplicar conceitos de design moderno
- Criar base reutilizável para projetos maiores

---

## 🧩 Funcionalidades

- 📷 Exibição de imagem do produto
- 📝 Nome e descrição
- 💰 Preço destacado
- 🛒 Botão "Order now"
- 🎨 Design com bordas arredondadas e sombra
- 📱 Estrutura pronta para expansão

## ▶️ Como executar

1. Clone o repositório:
git clone https://github.com/seu-usuario/coffee_card.git

2. Acesse a pasta:
cd coffee_card

3. Instale as dependências:
flutter pub get

4. Execute o projeto:
flutter run
